# Codex 微信桥接

这个上下文定义微信如何成为本机 Codex 的另一个入口，同时保持项目、会话、权限和历史的一致性。

## 入口与身份

**微信桥接**:
把微信入口连接到本机 Codex 的产品边界；它负责传递和路由交互，自身不是 Agent。
_Avoid_: 微信 Agent、Desktop 代理

**微信入口**:
同一 Codex 会话的另一种交互入口，不拥有独立的 Agent 身份或对话历史。
_Avoid_: 主 Agent、子 Agent、独立微信机器人会话

**微信控制者**:
唯一获准通过微信入口查看和操作本机 Codex 的人。
_Avoid_: 多用户共享、联系人默认授权

## 项目与会话

**Codex 项目**:
Codex Desktop 已保存工作区根目录所代表的工作空间；微信只显示根目录名称，完整路径仅用于内部路由。
_Avoid_: 任意会话目录、磁盘扫描结果、微信项目副本

**项目选择**:
微信控制者明确选中的 Codex 项目，用于浏览或新建会话。
_Avoid_: 隐式项目、自动猜测项目

**微信主会话**:
微信入口在没有进入项目会话时使用的默认持久化 Codex 会话。
_Avoid_: 默认项目、临时入口会话

**共享会话**:
由 Codex Desktop 和微信入口共同访问的同一个 Codex 会话，双方消息属于同一份持久化历史。
_Avoid_: 会话复制、微信镜像会话、独立微信会话

**微信回合**:
微信控制者通过微信入口提交到共享会话的一次 Codex 交互。
_Avoid_: 微信子会话、远程副本

**会话上下文**:
Codex 继续共享会话时使用的原有对话历史。
_Avoid_: 微信预览、截断副本

**会话预览**:
微信入口为识别共享会话而展示的有限摘要，不等同于会话上下文。
_Avoid_: 完整聊天导出、模型上下文、工具日志

**会话绑定**:
微信入口与某个共享会话之间的临时路由关系。
_Avoid_: 会话所有权、永久绑定

**通知回复窗口**:
主动通知与其来源共享会话之间的短期回复路由；它只在没有会话绑定且来源唯一时生效。
_Avoid_: 自然语言路由、永久自动进入

**选择快照**:
一次项目、会话或审批列表及其临时编号之间的固定映射。
_Avoid_: 永久编号、按新排序重新解释编号

## 执行与权限

**待处理消息**:
目标共享会话正在执行时，等待后续按顺序提交的微信消息。
_Avoid_: 插话、并发回合、改变当前回合

**会话权限模式**:
共享会话自身的 Sandbox、审批策略和审批者设置；它属于会话，而不属于微信入口。
_Avoid_: 微信权限、远程权限覆盖

**单次审批**:
某个微信回合为一项具体受限操作发出的临时授权请求。
_Avoid_: 永久授权、自动提权、跨入口审批

**中断回合**:
异常结束后未能正常完成的 Codex 回合。
_Avoid_: 可安全自动重试的消息、待处理消息

**状态未知**:
当前证据不足以判定回合已经完成、中断或仍在运行。
_Avoid_: 默认成功、默认重试

## 在场与通知

**在场状态**:
系统判定微信控制者正在电脑前使用当前 Windows 会话的状态。
_Avoid_: Codex 窗口焦点、Desktop 是否最小化

**离开状态**:
系统判定微信控制者未在电脑前使用当前 Windows 会话的状态。
_Avoid_: 仅窗口失焦

**微信回复**:
微信回合产生并返回微信的最终结果。
_Avoid_: 开始通知、推理直播、工具日志转发

**主动通知**:
由微信之外的本机 Codex 回合触发、主动发给微信控制者的状态消息。
_Avoid_: 微信回合的正常回复、在场时的重复通知

**未送达通知**:
已经产生但尚未被微信确认接收的回复或主动通知。
_Avoid_: 已丢弃通知、已读通知

## 事实源

**对话事实源**:
Codex 持久化会话中保存的会话历史；微信桥接不维护第二份对话。
_Avoid_: Bridge 聊天记录、微信历史副本

**传输状态**:
微信桥接为路由、去重和恢复而保存的短期记录，不属于对话事实源。
_Avoid_: 长期消息存档、复制 Codex 历史
